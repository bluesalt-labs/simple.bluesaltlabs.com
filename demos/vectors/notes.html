Creating Scalable Vector Graphics

The full Scalable Vector Graphics (SVG) 2 language specification is available at w3.org/TR/SVG2/Overview.html, but it's long and overwhelming. Below are notes taken for my reference while learning to build SVGs from a programmer's perspective.

https://www.w3.org/TR/SVG2/shapes.html
